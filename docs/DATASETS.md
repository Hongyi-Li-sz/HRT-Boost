# Datasets

This repository does not redistribute datasets. Place raw benchmark files in
`data/raw/` before running `scripts/run_full_benchmark.py`.

The default benchmark configuration expects the following files:

| Dataset | Expected file | Notes |
|---|---|---|
| Abalone | `abalone.data` | First column is categorical. |
| Kin8nm | `kin8nm.data` | Numeric regression dataset. |

Before public release, add the official source URL, license, and citation for
each dataset. Do not upload raw datasets unless their license explicitly permits
redistribution.
